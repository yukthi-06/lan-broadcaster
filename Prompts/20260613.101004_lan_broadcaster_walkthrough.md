# User Prompt
```
Create a complete desktop application in Lazarus (Free Pascal) called "LAN Broadcaster".

Requirements:

* Platform: Windows only.
* Framework: Lazarus LCL.
* Language: Object Pascal (Free Pascal).
* Produce a fully working Lazarus project with all units and forms.
* Use clean Object Pascal coding standards.
* Use classes and separation of concerns.
* Add comments explaining important sections.

Application Purpose:

The application allows any user on a local network (LAN) to send messages and file attachments to all other computers running the same application.

Networking Requirements:

1. Automatic Discovery

   * Use UDP Broadcast for discovery.
   * All running instances must automatically discover each other.
   * No manual IP entry.
   * Display online users in a list.
   * Show:

     * Computer Name
     * IP Address
     * Status (Online)

2. Messaging

   * Allow sending a text message.
   * Message is delivered to all discovered clients.
   * Display incoming messages in a chat/log window.
   * Include:

     * Timestamp
     * Sender Machine Name
     * Message Text

3. File Broadcasting

   * Allow selecting a file using OpenDialog.
   * Broadcast file metadata first.
   * Transfer files using TCP.
   * Support files up to at least 1 GB.
   * Transfer files in chunks (do not load entire file into memory).
   * Display transfer progress.
   * Save received files into:
     .\ReceivedFiles\
   * Automatically create the folder if it does not exist.

4. Reliability

   * Use TCP for messages and file transfers.
   * Use UDP only for discovery.
   * Handle disconnections gracefully.
   * Rebuild online user list automatically.
   * Prevent application crashes from network errors.
   * Add exception handling throughout networking code.

5. Multithreading

   * Networking must run in worker threads.
   * UI must never freeze.
   * Use TThread and thread-safe UI updates.
   * Use Synchronize or Queue when updating controls.
```

# LAN Broadcaster - Walkthrough

The "LAN Broadcaster" application is a complete, fully featured Delphi/Object Pascal desktop application targeting Lazarus LCL on Windows. It facilitates automatic user discovery, messaging, and multi-gigabyte file broadcasts.

## Changes Made

We created a modular project containing the following files:

1. [LANBroadcaster.lpr](file:///x:/Projects_X/0_Active/4_Lazarus/AI_Helped/LAN_Broadcaster_Lazarus_GIT/LANBroadcaster.lpr) - Main entrypoint file.
2. [LANBroadcaster.lpi](file:///x:/Projects_X/0_Active/4_Lazarus/AI_Helped/LAN_Broadcaster_Lazarus_GIT/LANBroadcaster.lpi) - Lazarus project layout setting dependencies on `indylaz` and `LCL`.
3. [NetworkTypes.pas](file:///x:/Projects_X/0_Active/4_Lazarus/AI_Helped/LAN_Broadcaster_Lazarus_GIT/NetworkTypes.pas) - Consts (`PORT_UDP_DISCOVERY = 5000`, etc.), types (`TOnlineUser`, `TTransferProgressRecord`), and events.
4. [Utilities.pas](file:///x:/Projects_X/0_Active/4_Lazarus/AI_Helped/LAN_Broadcaster_Lazarus_GIT/Utilities.pas) - Helper functions to obtain Machine Name, query Local IP Address, create `.\ReceivedFiles\` directory, and format file size strings.
5. [DiscoveryManager.pas](file:///x:/Projects_X/0_Active/4_Lazarus/AI_Helped/LAN_Broadcaster_Lazarus_GIT/DiscoveryManager.pas) - Handles UDP heartbeat broadcasts and listens for peers. Automatically prunes offline users after 15 seconds. Uses thread-safe notification wrappers to coordinate UI updates.
6. [MessageManager.pas](file:///x:/Projects_X/0_Active/4_Lazarus/AI_Helped/LAN_Broadcaster_Lazarus_GIT/MessageManager.pas) - TCP Server listening on port 5001 for text messages and a client thread to broadcast text messages without blocking the GUI.
7. [FileTransferManager.pas](file:///x:/Projects_X/0_Active/4_Lazarus/AI_Helped/LAN_Broadcaster_Lazarus_GIT/FileTransferManager.pas) - TCP Server listening on port 5002 for file downloads and client threads to stream uploads in 64KB chunks (perfect for files > 1 GB).
8. [MainForm.pas](file:///x:/Projects_X/0_Active/4_Lazarus/AI_Helped/LAN_Broadcaster_Lazarus_GIT/MainForm.pas) / [MainForm.lfm](file:///x:/Projects_X/0_Active/4_Lazarus/AI_Helped/LAN_Broadcaster_Lazarus_GIT/MainForm.lfm) - The user interface including side-by-side splitting panels, user lists, chat console, transfer log, progress records, and status bars.

3. Ensure the `indylaz` package is installed in your Lazarus package list.
