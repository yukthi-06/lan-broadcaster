# LAN Broadcaster Settings Feature Implementation Plan

We will add a settings section to the main window layout and implement system integration for startup behavior, tray minimization, sound alerts, and configuration persistence using an INI file.

## User Review Required

> [!IMPORTANT]
> - "Start on Windows Startup" writes to the current user's registry under `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`.
> - "Minimize on Close" and "Start Minimized" utilize a `TTrayIcon` component so the user can easily restore or terminate the application from the Windows system tray.
> - Sounds are played asynchronously using standard Windows Multimedia API (`PlaySound` from `mmsystem`).

- Play a notification sound when a message/file is received if `Enable Notification Sounds` is checked.

## Verification Plan

### Manual Verification
1. Open settings in Left Panel and verify changes persist across restarts (saved to `settings.ini`).
2. Enable "Start on Windows Startup", verify registry entry is created/removed when toggled.
3. Enable "Start Minimized", restart application, check if it starts directly in the system tray.
4. Enable "Minimize on Close", click the main window Close [X] button, check that the application remains running in the system tray. Restore it by double-clicking the tray icon.
5. Send a message from another client, verify a notification sound plays if checked, and is silent if unchecked.
