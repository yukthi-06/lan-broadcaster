# LAN Broadcaster Minimize on Close Fix Plan

We will resolve the issue where clicking the close button exits the application despite "Minimize on Close" being enabled. This occurs because the `FormCloseQuery` event handler declared in the code is not hooked up to the form's `OnCloseQuery` event in the `.lfm` definition.

## User Review Required

> [!IMPORTANT]
> - The `OnCloseQuery = FormCloseQuery` property will be added to the main form definition inside [MainForm.lfm](file:///x:/Projects_X/0_Active/4_Lazarus/AI_Helped/LAN_Broadcaster_Lazarus_GIT/MainForm.lfm).

## Proposed Changes

### [MODIFY] [MainForm.lfm](file:///x:/Projects_X/0_Active/4_Lazarus/AI_Helped/LAN_Broadcaster_Lazarus_GIT/MainForm.lfm)
Insert `OnCloseQuery = FormCloseQuery` near the top of the main form properties block.
